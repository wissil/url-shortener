Url Shortener help page

API endpoints

  • /account/: Otvaranje korisnickog racuna putem HTTP POST requesta.
  • /register/: Registracija URL-a putem HTTP POST requesta. Potrebna autorizacija korisnika.
  • /statistic/{accountId}: Statistika posjeta URL-ova registriranih od korisnika s predanim accountId.

Resources

  • /u/{urlId}: Redirekt klijenta na URL adresu konfiguriranu zadanim urlId.
